JSDoc

Reads — and writes — legacy Microsoft Word 97–2003 binary .doc files entirely in your browser: text, formatting, tables, lists, spacing, footnotes/headers, and images. Pure JavaScript, zero dependencies, clean-room from the [MS-DOC] / [MS-CFB] specs, 0BSD.

Drop a .doc here, or click to browse
Nothing is uploaded — extraction runs locally on this page.
100% client-side · no network 


    

    

      
      
      
      
    

  


  

    Source & tests:
    github.com/Alpaq92/JSDoc.
    Shows body text, footnotes/headers, character styling, tables, lists and embedded PNG/JPEG, with
    tracked deletions dropped; Download .doc writes a real Word file back with that
    formatting intact — styling, tables, lists, alignment, spacing/indentation, and inline images. It
    is not a pixel-faithful render — exact page layout and WMF/EMF images need a
    full engine — and it returns nothing for Word 6/95, encrypted, or non-.doc files
    (the host's download fallback).
